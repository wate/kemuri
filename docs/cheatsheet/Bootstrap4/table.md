[Bootstrap4(Table)](https://getbootstrap.com/docs/4.6/content/tables/)
=====================

table
---------------------

```html
<div class="table-responsive">
  <table class="table">
    <thead>
      <tr>
        <th scope="col">#</th>
        <th scope="col">First</th>
        <th scope="col">Last</th>
        <th scope="col">Handle</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <th scope="row">1</th>
        <td>Mark</td>
        <td>Otto</td>
        <td>@mdo</td>
      </tr>
    </tbody>
  </table>
  <!-- /.table -->
</div>
<!-- /.table-responsive -->
```
